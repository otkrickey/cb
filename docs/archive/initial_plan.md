# cb - macOS クリップボードマネージャー 初期仕様書

## プロジェクト概要

macOS標準のクリップボード機能を置き換える、高機能なクリップボードマネージャー。macOS標準のクリップボード履歴は使いにくいため、より直感的で生産性の高いクリップボード管理体験を提供する。

## 動機

macOS標準のクリップボード機能の課題:
- 履歴が1件のみ（直前のコピーのみ保持）
- テキスト・画像・ファイルの横断的な管理ができない
- 検索・フィルタリング機能がない
- ピン留め（お気に入り）機能がない

## コア機能

### クリップボード監視
- システムクリップボードの変更をリアルタイムで検知
- テキスト、画像、ファイルパス、リッチテキスト等の各種データタイプに対応

### 履歴管理
- コピー履歴の自動保存
- 履歴の検索・フィルタリング
- 履歴のピン留め（お気に入り）
- 履歴の手動削除
- 履歴の自動クリーンアップ（保持期間設定）

### UI/UX
- グローバルショートカットキーによるクイックアクセス
- Liquid Glass デザインシステムの採用（macOS Tahoe）
- テキストプレビュー・画像サムネイル表示
- ドラッグ&ドロップ対応

### ペースト操作
- 履歴からの選択ペースト
- プレーンテキストとしてペースト
- カスタムフォーマットでのペースト

## デザイン方針

- macOS Tahoe（macOS 26）で導入される Liquid Glass デザインを全面採用
- ネイティブmacOSアプリとしての統一感を重視
- 軽量・高速な動作を最優先

## アーキテクチャ

- **UIレイヤー**: Swift / SwiftUI（Liquid Glass デザイン）
- **ロジック・データレイヤー**: Rust（クリップボード監視、履歴管理、検索）
- **データベース**: SQLite（rusqlite、FTS5全文検索）
- **FFI**: swift-bridge（Swift ↔ Rust 連携）
- **ビルド**: Xcode + Cargo

## 技術要件

- macOS 26+ 対応
- Swift / SwiftUI（UI） + Rust（ロジック・データ）
- アクセシビリティ対応

## 配布方法

- GitHub Releaseからのインストーラー配布
- リポジトリからのソースビルド

## ターゲットユーザー

- macOSを日常的に使用する開発者・デザイナー・ライター
- クリップボード操作の頻度が高いユーザー
- macOS標準のクリップボード機能に不満を持つユーザー
