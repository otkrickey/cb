# usecases/ ディレクトリガイド

cbプロジェクト（macOSクリップボードマネージャー）のユースケースドキュメントを管理するガイド。

---

## 1. 目的

ユーザー（macOSユーザー）向けの操作ドキュメント。クリップボードマネージャーの操作方法を記述する。

**対象読者**: ユーザー（人間）

**文書化の方針**:
- アプリの操作手順を分かりやすく記述する
- 専門用語には簡潔な補足を添える
- スクリーンショットによる視覚的な補足を将来的に追加する

---

## 2. ディレクトリ構成

```
docs/usecases/
├── GUIDE.md                # 本ファイル
├── TEMPLATE.md             # 汎用テンプレート
└── (ユースケース追加時に作成)
```

---

## 3. 種別ごとのセクション取捨選択表

TEMPLATE.md の全セクションに対し、種別ごとの必須/任意/不要を示す。不要なセクションは削除して使用する。

| セクション | クリップボード操作 | 設定操作 |
|-----------|------------------|---------|
| 概要 | 必須 | 必須 |
| 前提条件 | 必須 | 必須 |
| 操作フロー | 必須 | 必須 |
| バリエーション | 任意 | 任意 |
| エラーケースと対処法 | 任意 | 任意 |
| 成功条件 | 必須 | 必須 |
| 関連ドキュメント | 必須 | 必須 |

---

## 4. 記法規約

### 操作フロー

ステップ番号付きで記述する。各ステップには期待結果を添える。

```markdown
1. ショートカットキー（⌘+Shift+V）を押す
   - **期待結果**: クリップボード履歴パネルが表示される
2. 貼り付けたい項目をクリックする
   - **期待結果**: 選択した項目がアクティブなアプリケーションに貼り付けられる
```

### スクリーンショット参照

```markdown
![クリップボード履歴パネル](./images/clipboard_panel.png)
```

### 相互参照

相対パスで参照する。

```markdown
- [クリップボードフロー](../design/flows/clipboard_flow.md)
```

---

## 5. テンプレートの使用方法

1. [TEMPLATE.md](./TEMPLATE.md) をコピーして新規ファイルを作成する
2. 種別に応じてセクション取捨選択表（3章）を参照し、不要セクションを削除する
3. 各セクションの内容を記述する
4. 空セクションが残っていないか確認する

### ファイル命名規則

| 種別 | 命名規則 | 例 |
|------|---------|-----|
| クリップボード操作 | `{操作名}.md`（小文字） | `clipboard.md` |
| 設定操作 | `{操作名}.md`（小文字） | `settings.md` |
