# レビュー出力形式

全モジュール共通の出力テンプレート。各モジュールの評価項目は `docs/review/modules/{module}.md` を参照。

## 出力形式

```markdown
## {Module} レビュー

### Critical（必須修正）
- ファイル:行番号 — 指摘内容
- （なければ「なし」）

### Medium（修正推奨）
- ファイル:行番号 — 指摘内容
- （なければ「なし」）

### Low（将来提案）
- ファイル:行番号 — 指摘内容
- （なければ「なし」）

### 確認済み
- 問題のなかった評価項目を列挙
```

## 記載ルール

- **指摘**: `ファイル:行番号 — 指摘内容` の形式で統一する
- **確認済み**: 評価項目のうち問題がなかったものを簡潔に列挙する
- 各レベルに該当がない場合は「なし」と記載する
