# 修正実行

プロンプトで渡された修正リストに従い、ファイルを修正する。

## 入力形式

修正リストは以下の形式で渡される:

```
1. [ファイルパス] 修正内容の説明
2. [ファイルパス] 修正内容の説明
...
```

## 手順

各修正項目について:

1. 対象ファイルを Read で読み、修正箇所のコンテキストを把握する
2. 行番号が不正確な場合は Grep/Glob で該当箇所を特定する
3. Edit で修正を適用する
4. 修正が意図通りか確認する

## 出力形式

```markdown
## 修正結果

### 完了
- [ファイルパス:行] 修正内容の要約

### 失敗（該当箇所が特定できない等）
- [ファイルパス] 理由
```
